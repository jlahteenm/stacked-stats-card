# Roadmap

This project is experimental. Features below are ideas, not commitments.

## Planned / Possible Enhancements

- [ ] Custom colors per entity
- [ ] Option to hide the legend
- [ ] Show totals per period
- [ ] Show averages per period
- [ ] Configurable period lengths
- [ ] Optional smoothing or rolling averages
- [ ] Translations (via Home Assistant localization)
- [ ] Compact mode
- [ ] Tooltip improvements
- [ ] Click-to-filter entity visibility
